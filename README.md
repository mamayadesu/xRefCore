# xRefCore
This framework will simplify a creation of PHP console applicaions with OOP using

<h2>Features:</h2>
* Class autoloading
* PHAR-packed<br>
* Windows Batch and Bash cyrillic support<br>
* Emulation of multithreading
* Asynchronous tasks
* Improved input-reading tools
* Pseudo GUI tools
* Build-in HTTP-server
